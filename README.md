# Lazo
Lazo plataforma móvil y web diseñada para coordinar la asistencia de pacientes mediante una red integrada de familiares, contactos de confianza y profesionales de la salud.

🚀 Descripción del ProyectoLazo centraliza la gestión del cuidado de pacientes en un solo lugar. La aplicación permite crear un ecosistema de colaboración en tiempo real donde los familiares gestionan el día a día, los contactos de emergencia reciben alertas críticas y los profesionales de la salud monitorean la evolución médica de forma remota y segura.✨ Características PrincipalesEcosistema de Roles: Accesos diferenciados para familiares (cuidadores principales), red de apoyo (vecinos/amigos) y médicos.Calendario Compartido: Coordinación de turnos de asistencia, visitas médicas y administración de medicamentos.Monitoreo de Signos Vitales: Registro histórico y gráficos en tiempo real de métricas clave (presión, glucosa, ritmo cardíaco).Módulo de Prescripciones: Alertas automatizadas de toma de medicamentos y control de stock de farmacia.Botón de Alerta SOS: Notificaciones geolocalizadas inmediatas a la red de contactos en caso de emergencia.Historial Clínico Seguro: Almacenamiento cifrado de recetas, reportes y diagnósticos accesibles para el personal autorizado.🛠️ Stack TecnológicoFrontend / Web: Next.js o React (desplegado en Vercel).Backend & API: Vercel Serverless Functions / Edge Functions.Base de Datos: PostgreSQL proporcionado por Supabase.Autenticación: Supabase Auth (Manejo integrado de roles de usuario).Almacenamiento: Supabase Storage (para recetas médicas, reportes y archivos).Tiempo Real: Supabase Realtime (para alertas SOS instantáneas y chat de red de apoyo).
